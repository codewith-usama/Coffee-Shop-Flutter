# Usage
Copy and paste below commands in terminal
<br/>
`git clone https://github.com/codewith-usama/Coffee-Shop-Flutter`
<br/>
`cd Coffee-Shop-Flutter`
<br/>
`flutter pub get`
<br/>
`flutter run`
